Student management system - 2023

Tecnologías: Java, Spring Boot, Oracle SQL.

He desarrollado una API RESTful utilizando Java y Spring Boot, que permite a los profesores gestionar sus estudiantes de manera eficiente. La API utiliza Oracle SQL como base de datos para almacenar y gestionar la información de los estudiantes. Los profesores pueden crear una cuenta en la plataforma utilizando su correo electrónico de Gmail y acceder a ella con su correo electrónico y contraseña correspondiente.

La API permite a los profesores crear nuevos estudiantes, ver información detallada de cada estudiante, actualizar la información existente de los estudiantes y eliminar estudiantes según sea necesario. Además, los profesores pueden organizar a los estudiantes en dos tandas, Matutina y Vespertina, lo que facilita la gestión de los estudiantes en diferentes momentos del día.

Sprite animation cutter - 2023

Tecnologías: Java, libGDX, Android Studio.

Sprite animation cutter es una aplicación móvil para Android desarrollada con Java, libGDX y Android Studio, diseñada para personas que hacen animaciones de sprites, esta aplicación es una mejora del Sprite animation player con algunas características nuevas, la aplicación te permite probar tus hojas de sprites, crear gifs animados a partir de una hoja de sprites o con sprites separados, y separar sprites de una hoja de sprites las cuales pueden ser exportadas como imagenes PNG.

La aplicación tiene más de 500 descargas con una calificación de 4.0⭐

Reading Flashlight - 2021

Tecnologías: Java, Android Studio.

Reading Flashlight es una aplicación móvil para Android desarrollada con Java en Android Studio, esta aplicación está diseñada para las personas que les gusta leer de noche, la aplicación te ayuda a leer libros, diarios o revistas gracias a su suave luz de la pantalla que es más agradable que la luz de la linterna de la cámara LED, el usuario puede cambiar el color y el brillo de la luz a su gusto.

La aplicación tiene más de 1000 descargas con una calificación de 4.5⭐
